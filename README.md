# Z’TITCH 👕🖤

Z’TITCH is a modern clothing brand e-commerce platform currently under development.

🚧 Status: Under Construction

## 🛠 Tech Stack (MERN)

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS

## 📌 Planned Features

- User authentication
- Product listing & filtering
- Shopping cart
- Secure checkout
- Admin dashboard
- Order management

## ⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/z-titch.git

Install backend dependencies:

cd server
npm install

Install frontend dependencies:

cd client
npm install

Run development servers:

npm run dev
